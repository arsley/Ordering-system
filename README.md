## Usage

### Setup

```shell
# copy .env.sample to .env
# then edit as you like it
$ cp .env.sample .env
```

### Run (on development)

```shell
$ make init
$ make up
```

### Run production

```shell
$ make production
```

### Run test

```shell
$ make rspec
```

### Generate ERD

```shell
$ make erd
```
